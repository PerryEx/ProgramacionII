Etiquetas en HTML y Codigo CSS

Algunas de las etiquetas mas usadas en HTML, se pueden apreciar a nivel de codigo y visual.

Etiquetas "h"

Las etiquetas h se utilizan para colocar titúlos. Hay 6 etiquetas las cuales con h1,h2,h3,h4,h5 y h6. La etiquita h1 se acostumbra a utilizar una sola vez para el titulo principal, la h2 y h3 se usan para poner mas de 2 titulo y las etiquetas h4 a la h6, indefinidamente.

Etiqueta "br" y Etiqueta "b"

La eitquera "br", nos sirve para decirle a la pagina que ahi termino el parrafo o linea y siga abajo.
como por Ejemplos, esto que acabamos de hacer.
La etiqueta "b" funciona para poder resaltar las palabras que deseemos como lo hicimos en esta fila.

Etiqueta "footer" y Etiqueta "aside"

La etiquera "footer", nos sirve para poder poner un pie de pagina y la Etiquita "aside" nos sirve para poder
indicar que se hara una barra lateral. En la pagina se puede apreciar esto.

Etiqueta "input" y Etiqueta "form"

Esta etiqueta nos permite decirle al programa que se crearan botones o textos
La etiqueta "b" nos sirve para almacenar ahi los bones o textos que usaremos.

Etiqueta "article"

Esta etiqueta nos permite poder agregar un articulo, como se muestra en esta pagina.

Etiqueta "ul y ol" y Etiqueta "li"

La etiqueta "ul" es por decirlo de una manera, donde se va a escribir la lista como si fuera un cuaderno.
Y la etiquera "li" es lo que contiene la lista. Ejemplo:


  • America
  • Europa
  • Africa
  • Oceania
  • Antartida

La lista que acabamos de ver es una lista desordenada "ul", para hacer una lista ordenada se utliza "ol".


  1. America
  2. Europa
  3. Africa
  4. Oceania
  5. Antartida
