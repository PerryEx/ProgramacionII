Etiquetas en HTML y Codigo CSS

Algunas de las etiquetas mas usadas en HTML, se pueden apreciar a nivel de codigo y visual.

Etiquetas "img"

Esta etiqueta nos permite poder mostrar una imagen en nuestra pagina.

Etiqueta "video"

Esta etiqueta nos sirve para poder mostar un video en nuesta pagina.

Etiqueta "audio"

La etiqueta audio sirve para que se pueda escuchar un audio.

Etiqueta "header"

Esta etiqueta nos se usa para poner un encabezado como se meustra en estas dos paginas, las cuales es el titulo principal.
